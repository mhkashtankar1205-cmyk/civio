# Civio

Civio is a community-driven civic issue reporting platform that connects citizens with municipal authorities.

## Features

### Citizen Portal

* Report civic issues with images
* GPS-enabled location reporting
* Support community issues
* Distance-based issue feed
* Notifications
* Multilingual support (English, Hindi, Marathi)

### Municipal (MNC) Dashboard

* Admin-only access
* Issue workflow management
* Status transitions:

  * Pending
  * Under Review
  * In Progress
  * Resolved
* Before and after image verification
* Trending issue prioritization
* Resolution tracking

## Tech Stack
